# english_words_3000

日常でよく使われる約3000の英単語が、どのくらいわかっているか隙間時間を使って確認できるアプリです。

<br />
<br />

・目的<br />
日常英会話でよく使われる約3000語をどのぐらい理解できているかを素早く確認。
隙間時間を活用。
